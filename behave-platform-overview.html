01 — PIPELINE

How a simulation run works

One creative in → one decisive recommendation out. Every step is auditable.

📄 Creative Card → 👥 Cohort (20) → 🧠 Persona Eval ×20 → 🛡️ Critic Gate ×20 → 📊 Scorecard → 🎯 Recommendation

Per Persona (Haiku — $0.005/eval)

Retrieve top-5 memories → Build rich prompt (persona + creative + memory context) → LLM generates structured evaluation in-character (scores, reasoning, verbatim reaction)

Critic Gate (Sonnet — $0.012/check)

Scores on 4 dimensions: persona consistency, sycophancy detection, cultural authenticity, action-reasoning alignment. FAIL → quarantine. Only genuine evaluations aggregate.

Memory Formation

Score > 70 → PREFERENCE node. Score < 30 → CATEGORY_BELIEF node. Cumulative importance ≥ 50 → Reflection triggers (higher-order insight synthesis).

04 — MEMORY

Long-Horizon Memory Retrieval

Based on Park et al.'s Generative Agents — personas carry persistent memories across simulation runs. A persona who distrusted a brand last month will carry that skepticism forward.

score  =  α · recency  +  β · relevance  +  γ · importance
RECENCY

0.99 ^ hours_elapsed

Exponential decay. Memory from 3 days ago scores 0.49. From 30 days = 0.0007. Recent experiences dominate.

RELEVANCE

cosine_sim(query, memory)

Semantic match. Falls back to keyword overlap: text_overlap × 0.7 + theme_overlap × 0.3 when embeddings unavailable.

IMPORTANCE

poignancy / 10.0

LLM-scored memorability. A traumatic brand experience (8/10) persists longer than a forgettable ad (2/10).

4 Memory Types

OBSERVATION — raw eval reaction PREFERENCE — score > 70
CATEGORY_BELIEF — score < 30 REFLECTION — synthesized insight

Config

α = β = γ = 1.0 decay = 0.99 top_k = 5 reflection threshold = 50
